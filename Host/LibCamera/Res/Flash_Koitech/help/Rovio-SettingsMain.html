Settings - Overview

This section provides and overview of Rovio's Settings. For more specific help click on the help icon in the selected tab.

Tab Description

Video tab

Allows you to change Rovio's video settings (e.g. resolution, video quality, brightness, framerate, etc).

You information about audio support in Internet Browsers apart from Internet Explorer click here.

Movement tab

Allows you to change Rovio's movement settings (e.g. movement speed, IR radar, etc).

Photo tab

Allows you to change settings for emailing screen captures from Rovio’s interface.

Security tab

Allows you to change Rovio's security settings.

Network tab

Allows you to change Rovio's Wi-Fi and network options and settings.

Domain tab

Allows you to setup a domain to access Rovio.

System tab

Allows you to reboot, restore default settings, and update Rovio's firmware.
